Installation instructions

Below is the Google tag for this account. Copy and paste it in the code of every page of your website, immediately after the <head> element. Don't add more than one Google tag to each page.

<!-- Google tag (gtag.js) --> <script async src="https://www.googletagmanager.com/gtag/js?id=G-5FH5V676CW"></script> <script> window.dataLayer = window.dataLayer || []; function gtag(){dataLayer.push(arguments);} gtag('js', new Date()); gtag('config', 'G-5FH5V676CW'); </script>
